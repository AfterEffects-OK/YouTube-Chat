\# カスタム YouTube Live Chat Viewer



\## 概要



これは、YouTube LiveのチャットをOBSなどの配信ソフトで表示するために設計された、高度にカスタマイズ可能なチャットビューアです。ローカルのHTMLファイルとして動作し、ブラウザソースとしてOBSに追加するだけで、デザインを自由に変更できるチャット欄を配信画面に表示できます。

\## 主な機能



\### 1. 柔軟なレイアウトとデザイン設定

\- \*\*フォント設定\*\*: 全体、ユーザー名、チャット本文、さらにピン留めエリア専用のフォントを個別に設定可能。

\- \*\*サイズと色\*\*: ユーザー名とチャット本文の文字サイズ・色をそれぞれ独立して調整できます。

\- \*\*表示モード\*\*: チャットを1行で表示するか、名前と本文を分ける2行表示にするかを選択できます。

\- \*\*インデントと間隔\*\*: 2行表示時のインデントや、名前と本文の間の余白を細かく調整できます。

\- \*\*背景画像\*\*: チャットメッセージの背景に好きな画像（GIFアニメーションも可）を設定できます。



\### 2. 高度なチャット表示機能

\- \*\*スーパーチャットストック\*\*: 受信したスーパーチャットを右側の専用エリアにストックし、見逃しを防ぎます。

\- \*\*チャットピン留め機能\*\*:

&nbsp;   - 通常チャット、SCストック内のチャットをクリックすると、画面下部の固定エリアに最大6つまでピン留めできます。

&nbsp;   - ピン留めしたチャットを再度クリックすると、アニメーション付きで解除されます。

\- \*\*多彩なチャット表現\*\*: 通常チャット、モデレーター、メンバー、スーパーチャット（金額表示付き）、スーパーステッカーをそれぞれ異なるスタイルで表示します。



\### 3. 配信者向けユーティリティ

\- \*\*設定の永続化\*\*: すべてのデザイン設定とAPIキーはブラウザのローカルストレージに保存され、次回起動時も同じ設定で利用できます。

\- \*\*デザインテスト\*\*: 「デザインテスト」ボタン一つで、全種類のダミーチャットを生成し、リアルタイムでデザインを確認できます。

\- \*\*クリア機能\*\*: 「全クリア」ボタンと、ピン留めエリアのみをクリアする「ピンのみクリア」ボタンを搭載。

\- \*\*分かりやすい通知\*\*: 設定の保存完了やAPIエラーなどを、画面を邪魔しないトースト通知でお知らせします。



\## 使い方



\### 1. 準備するもの

\- \*\*YouTube Data API v3 キー\*\*: Google Cloud Consoleから取得してください。

&nbsp;   1. \[Google Cloud Console](https://console.cloud.google.com/)にアクセスします。

&nbsp;   2. 新しいプロジェクトを作成します。

&nbsp;   3. 「APIとサービス」 > 「ライブラリ」で「YouTube Data API v3」を検索し、有効にします。

&nbsp;   4. 「APIとサービス」 > 「認証情報」で「認証情報を作成」 > 「APIキー」を選択してキーを生成します。



\### 2. セットアップ

1\.  このプロジェクトの `index.html` ファイルをPCに保存します。

2\.  `index.html` をGoogle Chromeなどのウェブブラウザで開きます。

3\.  画面左側の「ライブチャット設定」欄に、取得した\*\*APIキー\*\*と、チャットを取得したい\*\*YouTube動画のIDまたはURL\*\*を入力します。



\### 3. デザインのカスタマイズ

1\.  「デザインカスタム」セクションのアコーディオンメニュー（背景画像、レイアウト、フォント設定、サイズと色）を開き、好みのデザインに調整します。

2\.  「デザインテスト」ボタンを押すと、ダミーのチャットが表示され、変更をリアルタイムで確認できます。

3\.  設定が完了したら、「設定を保存」ボタンを押してください。次回以降、同じ設定が自動で読み込まれます。



\### 4. OBSへの取り込み

1\.  OBSで、チャットを表示したいシーンに「ブラウザ」ソースを追加します。

2\.  プロパティ画面で「ローカルファイル」にチェックを入れ、「参照」ボタンから先ほど保存した `index.html` ファイルを選択します。

3\.  幅と高さをOBSのキャンバスサイズに合わせて設定します（例: 幅: 1920, 高さ: 1080）。

4\.  「OK」を押すと、カスタマイズしたチャットビューアがOBSの画面に表示されます。

5\.  ブラウザで「チャット取得開始」ボタンを押すと、OBS上の表示も連動してチャットの取得が始まります。



\## 注意事項

\- YouTube Data APIには1日あたりの利用上限（クォータ）があります。個人での利用範囲では通常問題ありませんが、長時間の配信や頻繁なリロードを行うと上限に達する可能性があります。

\- このツールはローカルファイルとして動作するため、インターネット接続が必要です。
